# 快速开始

欢迎来到已经配置好的 Nextra 文档站点。所有文档都位于 `src/content` 目录下的 `.mdx` 文件中。

## 目录结构

```text
src/
  app/            // Next.js App Router 页面
  content/        // 文档内容
mdx-components.js // 可在这里扩展自定义组件
```

## 常用命令

- `npm run dev`：启动本地开发服务器
- `npm run build`：构建生产版本
- `npm run start`：预览生产构建

现在可以在侧边栏中打开“示例文档”来查看更多内容。
