# 示例文档

这个页面演示了 Nextra 的一些内置组件，以及如何同时使用 Markdown 与 JSX。

<Callout type="info">
  将新的 `.mdx` 文件放入 `src/content` 目录即可立刻生成新的文档页面。
</Callout>

## 下一步

1. 在 `_meta.js` 中管理侧边栏与文档顺序。
2. 新增的 `.mdx` 文件会自动出现在左侧导航。
3. 根目录的 `next.config.mjs` 可以调整 Nextra 行为，比如禁用代码复制按钮等。
